# gbmd-mdoc/astro
An astro plugin to convert synced [Gitbook's](https://gitbook.com) markdown syntax to [Markdoc's](https://markdoc.dev)

## Usage

```bash
gbmd-mdoc -i inputDir -o outputDir

```
